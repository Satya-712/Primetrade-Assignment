# Trader Behavior Analysis Based on Market Sentiment

## Objective

Analyze the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using Hyperliquid historical trading data.

## Datasets Used

1. Bitcoin Fear & Greed Index Dataset
2. Hyperliquid Historical Trader Dataset

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* NumPy

## Methodology

1. Data loading and cleaning
2. Date conversion and preprocessing
3. Dataset merging using trade date
4. Exploratory Data Analysis (EDA)
5. Profitability analysis by sentiment
6. Buy vs Sell performance comparison
7. Top trader and top coin analysis

## Key Findings

* Fear sentiment showed the highest trading activity.
* Extreme Greed delivered the highest average profit per trade.
* Fear generated the highest overall profit.
* SELL trades outperformed BUY trades.
* A small number of traders generated a large portion of total profits.
* HYPE, SOL, ETH and BTC were among the most profitable assets.

## Conclusion

Market sentiment significantly influences trader behavior and profitability. Fear periods attract higher participation, while Extreme Greed periods provide the highest average profitability. Monitoring sentiment indicators can help traders improve risk management and trading decisions.
